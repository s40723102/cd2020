  • IP 查詢

    ifconfig

    如果跳出尚未安裝net-tools的話請安裝

    sudo apt install net-tools

    如果在IPv6環境下有跳出網路錯誤

    1.IPv6環境設定是否正確

    到/etc/netplan 目錄下檢查 00-installer-config.yaml 檔案內的設定
    1.cd /etc/netplan
    2.sudo vi 00-installer-config.yaml
     00-installer-config.yaml(按照格式打)

  • 編輯完成後:wq跳出編輯並儲存，更新netplan設定

    sudo netplan apply

  • proxy問題
    到 /etc/apt/apt.conf.d 目錄下檢查是否有proxy.conf檔案，若沒有，請用新增    cd /etc/apt/apt.conf.d
    sudo vi proxy.conf
  • proxy.conf檔案的內容

    Acquire::http::proxy "http://[2001:288:6004:17::填port號]:3128";

  • 更新aptsudo apt update
  • 完成即可啟動wsgi.py
